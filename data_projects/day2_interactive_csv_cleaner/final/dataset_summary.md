Final Dataset Summary

Source: House Pricing

Cleaning Steps:
* Missing numeric values filled with column median
* Missing boolean values filled with False
* Missing Categorical values filled with "unknown"

Columns:
* Property Name: Name of property
* Area with Type: Type of area
* Square Feet: Square feet occupied by the property
* Transcaction: New property or Resale
* Status: Ready to move or possesed
* Floor: Floor where it is located
* Furnishing: Unfurnished, Semi-furnished
* Facing: Which direction it's facing
* Description: How the property looks
* Price per sqft: Price per sqft
* Price: Price of the property

Notes:
This dataset represents a finalized, cleaned version suitable for analysis and visualization.
